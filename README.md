# pokedex
Pokedex CLI

Functionalities:
 - map pokemon areas
 - explore especific locations
 - catch pokemon
 - see catched pokemon data
 - pokemon starter 
 - pokemon teams 
 - pokemon battles
 - random encounters
 - pokemon level up
 - see your team and their stats
 - command change team
